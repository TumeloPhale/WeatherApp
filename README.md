# WeatherApp

A C# desktop application that fetches and displays current weather data for a given location.

Built during university as one of my first real projects — kept here as part of the journey.

---

## What it does

- Accepts a city name as input
- Fetches current weather from a public API
- Displays temperature, conditions and basic forecast

## Tech

- C# / .NET
- HTTP client for API calls
- WinForms UI

## Running it

```bash
git clone https://github.com/TumeloPhale/WeatherApp.git
# Open in Visual Studio, add your OpenWeatherMap API key, then F5
```
